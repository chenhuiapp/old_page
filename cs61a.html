Data Abstraction

Data Abstraction is a very important concept, since it helps programmers to treat code as objects. An abstract data type consists of two types of functions:
  • Constructors: Functions that will build the abstract data type.
  • Selectors: Functions that will retrieve information the abstract data type store.

Sequence

A sequence is an ordered collection of values. The sequence is a powerful, fundamental abstraction in computer science. Sequences all share two common behavior
  • Length: A sequence has a finite length. An empty sequence has length 0.
  • Element selection: A sequence has an element corresponding to any non-negative integer index less than its length, starting at 0 for the first element.
